System Architecture

Three operation entries, one system

The base station carries edge computing — so stores keep operating even offline, while the cloud platform manages every store centrally.

Local · Port 80 Base-station business system Labels, products, templates, bind/unbind, refresh, roll-call — runs on the gateway itself, works offline for on-site debugging and emergencies.
Local · Port 90 Base-station backend Server and network configuration, key management, label and base-station firmware upgrades, system settings.
Cloud · WEB Central management platform Multi-store base-station registration, label and template management, product binding, and data dashboard — the main entry for daily operations.
